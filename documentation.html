  11. API integration: AiGURU integrates with OpenAI's API, allowing users to access and utilize the latest AI technologies and models.
  12. Multiple output formats: The app can generate content in a variety of output formats, including text, images, and documents, making it a versatile tool for a wide range of use cases.
  13. Category and tag system: The app includes a robust category and tag system, making it easy to organize and find content.
  14. Advanced search: AiGURU includes an advanced search system that allows users to quickly and easily find specific content based on keywords, tags, and other criteria.

Requirements for AiGURU System

In this section we will present the recommended configuration under which AiGURU would work perfectly. These are not absolute values as the performance depends on many factors such as host performance, server configuration and the number of visitors.

Server requirements

  • Apache or IIS or Nginx
  • PHP version 8.1.0 >= 8.1.0 (ie. recommended: Any PHP version greater or equal than 8.1.0 )
  • MySQL version 5.0 or greater (recommended: MySQL 5.5 or greater)
  • We recommend DigitalOcean for hosting. Or VPS Server for smooth and best performance


AiGURU Installation

  1. Please extract the AiGURU_Master-v1.0.0.zip folder. You will then get the following:

    • AiGURU_Master-v1.0.0.zip
    • Documentation (Folder)

    We highly recommend that you read the documentation first.

  2. Upload the AiGURU_Master-v1.0.0.zip file to your server using cPanel or FTP.
  3. Extract the AiGURU_Master-v1.0.0.zip file.
  4. Navigate into the extracted folder and then move all the folders and files present. make sure those files and folders are inside the public_html folder
    • app
    • bootstrap
    • config
    • database
    • lang
    • modules
    • public
    • resources
    • routes
    • sql
    • storage
    • vendor
    • .env.example
    • .htaccess
    • artisan
    • composer.json
    • composer.lock
    • modules_statuses.json

    CAUTION: Make sure that the .htaccess file is placed correctly or else you will be unable to access the admin panel.

    5. NOTE: The instructions above will be applicable for all other system!

  5. To setup your application, browse:

    http://yourdomain.com/install/index.php

    Then enter your Envato User ID and Purchase Key. (NOTE: Your purchase key can be found at your envato download option: https://themeforest.net/downloads) Once the form has been filled, click on the Next button.

  6. Installer second step checks to see whether your directory and server meet the application installation requirements.
    NOTE: SMTP server must be enabled to send emails. However, it is not mandatory during the installation process. You will have to setup your SMTP server later otherwise your emails will fail to deliver.


    7. IMPORTANT NOTE: If you see errors in "Directory permissions" due to files not being writable as shown in the screenshot below then please change their read-write permissions 755 or higher in the file properties as guided by the screenshots below.

    • Go to storage folder (Location: /home/your_site_name/public_html/storage)

      Click the specific folder and got to Properties or Folder Permission option

    • Then Add Permission 0755 or higher( Owner:read,writable,execute | Group:read,execute | User:read,execute ) and click on the Change Permission/Okay button.


  7. Environment Requirements.



    8. IMPORTANT NOTE: If you see errors in ".env not found" error as shown in the screenshot below then click on the Create ENV File button to create the .env file and wait 4-5 seconds for the page to refresh.



    IMPORTANT NOTE: If you see errors in "Vendor not found" error as shown in the screenshot below then you will have to ssh access to your server and run the following command in the root directory of your application.

    Command: composer install --optimize-autoloader --no-dev

    Then refresh the page. If the error still persists then you will have contact our support team to get this issue resolved.



  8. Database Configuration. Go to your database manager and create a database and create a user for maintaining the database. Then enter the database name, database username, database password, database host and database port. Then click on the Next button.

    Note: the database must be blank and the user must have all privileges to the database.


  9. Now Create Admin User. Enter your email, password and click on the Submit button. then wait 4-5 seconds for the page to refresh.

    Note: the admin username and password must be unique. if you see any error then refresh the page and try again. If the error still persists then you will have contact our support team to get this issue resolved.


  10. Installation successfully done! Thank you for purchasing AiGURU. If you have any questions, please contact us at info@bdtask.com.

    Optimization Tips

    Note: If you have SSH Access/Terminal access then you can take the following steps to make your application faster and optimized.

    1. Open Your Terminal and go to your project folder Like "cd your_project_path".




    2. Then Run This Command "php artisan storage:link" to connect storage folder symbolic to public.




    3. Run This Command "php artisan key:generate" to generate new encryption key for security purposes.




    4. Run This Command "php artisan event:cache && php artisan view:cache && php artisan route:cache && php artisan config:cache" to cache configuration,route and view files.



    5. Note: If you fetch any issue/broken your application after optimize/cache command run then you can clear cache by run this command "php artisan optimize:clear".



    Open Ai Configuration

    After a successfully installation let's configure the App. Let's explore the crazy things.

  • OpenAI API Credentials Configuration

    • IMPORTANT NOTE: To use automation modules in Ai Guru application, you must first configure Open-Ai's API in the application. So without further delay let's get started.

    1. Go to Settings from the left side bar menu, click on Open Ai Setting from the Settings menu. After that fill all the credentials of your "Open-Ai Configuration"

      • Fill the OpenAI API Key input field.
      • Fill the OpenAI Organization Key input field.
      • Fill the OpenAI Max Number of Result (1-10) field.
      • Fill the OpenAI Max Token (100-2000) field.

      If you haven't any OpenAI API credentials yet please purchase it from OpenAI.

    Open-Ai Instruction

    1. First you need to register on Open-Ai's website. After that you need to login to your account.
    2. After successful login, click on your picture on the top right corner of your dashboard and click on the View API keys button.
    3. Now Click "Create new secret key" button To create new api secret.Enter your application name then click "Create secret key" button
    4. Copy Your app secret key and paste it in our application the "OpenAI API Key" field in the application.
    5. Then go to Organization tab and copy your organization id and paste it in our application the "OpenAI Organization" field in the application.

    Google Invisible Recaptcha Configuration

    IMPORTANT NOTE: In this Application system Invisible reCaptcha API is used for security purpose. Invisible reCAPTCHA uses an advanced risk analysis engine and adaptive challenges to keep malicious software from engaging in abusive activities on your website. Meanwhile, legitimate users will be able to login, make purchases, view pages, or create accounts and fake users will be blocked.



      Invisible reCAPTCHA Instruction

      1. To use Invisible reCAPTCHA badge first visit this link.
      2. After that give your application name in "Label". And in reCAPTCHA type Select (Challenge (v2)->Invisible reCAPTCHA badge).
      3. Enter all your domains one by one in your Domains list and press enter
      4. Finally click on "Submit" button
      5. Now copy the site key and paste it in the reCAPTCHA Site key and copy the secret key and paste it in the reCAPTCHA secret key.
